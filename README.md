# Thorpe Park – Guía Informativa No Oficial

Página web estática e informativa sobre Thorpe Park Resort. **No es la página oficial ni está afiliada a Merlin Entertainments.**

## Archivos
- `index.html` — Página principal completa
- `style.css` — Todos los estilos
- `script.js` — Interacciones (menú, filtros, FAQ, animaciones)
- `vercel.json` — Configuración de despliegue para Vercel

## Desplegar en Vercel (3 pasos)

### Opción A – Arrastrar y soltar (más fácil)
1. Ve a [vercel.com](https://vercel.com) e inicia sesión
2. En el dashboard, haz clic en **"Add New → Project"**
3. Arrastra la carpeta `thorpe-park` directamente a la ventana
4. Vercel detecta que es estática y la despliega automáticamente ✅

### Opción B – Desde GitHub
1. Sube la carpeta a un repositorio de GitHub
2. En Vercel, importa el repositorio
3. Framework preset: **Other** (sin framework)
4. Build command: *(dejar vacío)*
5. Output directory: *(dejar vacío o `.`)*
6. Haz clic en Deploy ✅
